Esse repositório Contém dois arquivos, um realizado com o site Books to scrape e o outro realizado com o site Hacker News.

Os dois códigos são similares e utilizam do mesmo requirements.txt.

Livros:
Um script de Python que Utiliza BeautifulSoup e Requests + Funções built in do python para realizer Web Scraping de Livros no Site https://books.toscrape.com

Tenha em Mente que o Books To Scrape utiliza dados completamente ficticios.

Você escolhe uma nota de 1 a 5 e o script vai percorrer as primeiras 10 páginas do Site e buscar todos os livros com a nota selecionada, depois te recomendará um deles aleatóriamente.

Como Utilizar:

Clone o Repositório e realize a instalação dos Requirements 
pip install -r requirements.txt

Execute o Arquivo biblioteca.py

Selecione uma Nota de 1 a 5

Veja a magia acontecer.

Noticias:
Script de Python que pega dados do site https://news.ycombinator.com/ e busca noticias por pontuação minima da noticia, como o site possui várias
noticias de pontuações muito variadas baseadas na avaliação dos usuários é mais dificil de separar por pontuação exata (como é feito no script dos livros).

Como Utilizar:

Clone o Repositório e realize a instalação dos Requirements
pip install -r requirements.txt

Execute o Arquivo jornal.py

Selecione a pontuação minima das noticias

Apenas espere o Jornaleiro te entregar sua noticia do dia.
